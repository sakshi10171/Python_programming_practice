# Python File Handling

This folder contains **Python programs demonstrating file handling operations**.  
These programs cover basic to intermediate concepts such as reading, writing, appending, searching, and deleting files.

---

## 📂 Programs Included

| File Name | Description |
|----------|-------------|
| write_file.py | Writes data to a file |
| read_file.py | Reads data from a file |
| word_count_file.py | Counts number of words in a file |
| append_file.py | Appends data to an existing file |
| line_count.py | Counts number of lines in a file |
| search_word.py | Searches a word in a file |
| copy_file.py | Copies content from one file to another |
| delete_file.py | Deletes a file |

---

## 📌 Concepts Covered
- File creation and writing  
- File reading  
- Appending data  
- File copying  
- Searching data in file  
- Counting words and lines  
- File deletion  
- Use of `with open()` for file handling
