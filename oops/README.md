# Python OOP Programs

This folder contains **Object-Oriented Programming (OOP) concepts in Python**.  
These programs demonstrate basic to intermediate OOP concepts with simple examples.

---

## 📂 Programs Included

| File Name | Description |
|----------|-------------|
| class_object.py | Demonstrates class and object |
| encapsulation.py | Demonstrates data hiding using private variables |
| inheritance.py | Demonstrates inheritance |
| polymorphism.py | Demonstrates method overriding |
| abstraction.py | Demonstrates abstraction using abstract class |
| student_management.py | Real-world example using class, constructor, and methods |
| multiple_inheritance.py | Demonstrates multiple inheritance |

---

## 📌 Concepts Covered
- Class and Object  
- Constructor (`__init__`)  
- Encapsulation  
- Inheritance  
- Polymorphism  
- Abstraction  
- Multiple Inheritance
